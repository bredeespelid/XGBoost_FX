# MLBenchmarks_FX

This repository contains code and notebooks used for the master's thesis work on forecasting foreign-exchange levels using machine learning methods.

- Purpose: Benchmarks for EUR/NOK forecasting using Gradient Boosting models and macro covariates.
- Contents: Jupyter notebooks are in the `src/` folder; data files are in the `data/` folder.
- Status: Supporting material for a master's thesis (research use).
